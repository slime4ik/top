О платформе Роли Задачи Группы Чат Правила

Taskly — управление задачами для команд

1

Платформа для совместной работы над задачами с лимитом 50 участников на группу

2

Простая система делегирования и отслеживания прогресса

Роли участников

Создатель
  • Создает группу
  • Назначает админов
  • Управляет настройками
Администратор
  • Управляет задачами
  • Принимает участников
  • Модерирует контент
Участник
  • Берет задачи
  • Общается в чате
  • Отслеживает прогресс

Как работают задачи

Взятие задач

Любой участник может взять задачу. Ограничений по количеству нет — берите столько, сколько сможете выполнить.

Статусы

Ожидание → В работе → Выполнено. Можно создавать подзадачи для сложных проектов.

Типы групп

Открытые
  • Доступ для всех
  • Видимость публичная
  • До 50 участников
Закрытые
  • По приглашению/коду
  • Видимость приватная
  • До 50 участников

Групповой чат

В каждой группе есть свой онлайн-чат:

  • Общение в реальном времени
  • Обсуждение задач и проектов
  • Уведомления о новых сообщениях
  • Доступ только для участников группы

Основные правила

  1. Берите только посильные задачи
  2. Соблюдайте сроки выполнения
  3. Уважайте других участников
  4. Не передавайте коды доступа
  5. Сообщайте о проблемах админам
